# hurriyet-ruby
Unofficial Ruby Wrapper for Hurriyet API - http://developers.hurriyet.com.tr/

# Getting Started
`client = Hurriyet::Client.new(<YOUR_API_KEY>)`

# Resources
- [Articles](#articles)

## Articles

### Get all articles
`client.articles.all`

### Get a single article
`client.articles.single id: 40220736`
